# SkinShark Partner Docs

Merchant and partner API documentation for SkinShark, built with [Mintlify](https://mintlify.com).

## Development

Preview changes locally:

```bash
npm i -g mint
mint dev
```

View at `http://localhost:3000`.

## Validation

```bash
npx --yes mint validate
npx --yes mint broken-links
```

## Deploying

Changes pushed to the default branch are deployed automatically via the Mintlify GitHub app.
